# Carnatic Analyzer

An interactive Carnatic music analysis platform built with React, TypeScript, Vite, and Tailwind CSS.

## Features
- Raga Identifier
- Raga Browser
- Tala Reference
- Pitch Detection
- Composition Assistant
- Gamaka Explorer
- Dark Mode Support

## Live Demo
https://carnatic-analyzer-carnatic-analyzer.vercel.app

## Technologies
- React
- TypeScript
- Vite
- Tailwind CSS
- Wouter
- Lucide Icons
